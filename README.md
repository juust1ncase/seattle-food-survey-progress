# Seattle Survey GitHub Pages Upload

1. Go to github.com
2. Click + > New repository
3. Name it something like `seattle-industry-survey`
4. Make it Public
5. Click `uploading an existing file`
6. Upload `index.html`
7. Commit changes
8. Go to Settings > Pages
9. Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
10. Save

Your site will publish at:
https://YOURUSERNAME.github.io/seattle-industry-survey/
